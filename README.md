# schobidotdev Claude Plugins

Personal plugin marketplace for Claude Code.

## Plugins

| Plugin | Type | Description |
|---|---|---|
| [discord-focus](external_plugins/discord-focus/) | MCP | Discord channel with per-session channel focus filtering |
| [companion](external_plugins/companion/) | Hooks | AI session companion with personality, voice, smart summaries, and notifications |

## Installation

```
/plugin install discord-focus@schobidotdev
/plugin install companion@schobidotdev
```
